# Faraway

[repository](https://github.com/kytware/faraway)

![reflections](https://raw.githubusercontent.com/kyteware/faraway/23cc8e2fbec3550262aa230cdcfabb2dadb38fbe/examples/mirror_room/mirror_room.png)

I wrote Faraway for a Java class project in high school!

It is a raytracing renderer featuring:
- Coloured lights
- Shadows
- Reflections
- A scene file format (using JSON)
